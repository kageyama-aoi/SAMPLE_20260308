auto_awesome テキストを解析する
  1. 代行依頼のメッセージをテキストエリアにコピペ
  2. 「解析する」ボタンを押す
  3. 解析結果を確認・修正して「すべて登録する」
smart_toy うまく解析できない場合
  1. 「うまく解析できない場合」をクリック
  2. 「プロンプトをコピー」でコピー
  3. ChatGPT や Claude に貼り付けて解析してもらう
  4. 結果の JSON を「インポート」から取り込む
table_rows 一覧テーブル
  • 列ヘッダーをクリック → ソート（▲▼）
  • 「未決のみ表示」チェック → フィルター
  • 依頼者・代行者欄は直接入力で編集可
  • ステータスバッジをクリック → 未決 ↔ 対応済み
  • delete ボタンで削除
file_download データのバックアップ
  • 「エクスポート」→ JSON ファイルとして保存
  • 「インポート」→ JSON ファイルを読み込んで追加
  • データはブラウザ内（localStorage）に保存されます
calendar_month カレンダービュー
  • タブで「月カレンダー」「週カレンダー」を切替
  • ◀ ▶ で月・週を移動
  • チップにマウスを乗せると詳細が表示
  • 週カレンダーは午前 / 昼間 / 夜間 で分類表示
